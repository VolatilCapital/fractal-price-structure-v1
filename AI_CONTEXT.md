# AI_CONTEXT — fractal-price-structure

> Companion to `README.md`. Concise reference for AI assistants. Generated by Magic Position ecosystem audit (2026-05-13).

## Position in the ecosystem
- **Layer:** Standalone — not in MAGIC_POSITION's direct dep tree as of 2026-05-13.
- **Role:** TypeScript library for generating hierarchical fractal price structures from candlestick (OHLCV) data with recursive PriceMove analysis.
- **Distributed via:** `github:VolatilCapital/fractal-price-structure#3f3ca35`
- **Consumed by:** Unknown — no consumers identified in the audited scope.

## Public API
- **@fractal-price-structure/core** — `FractalEngine`, `ConsoleLogger`, PriceMove analysis (fractal pattern detection)
- **@fractal-price-structure/visualizer** — Web UI + terminal demo for fractal visualization

## Allowed dependencies
- **Runtime:** `axios`, `big.js`, `dukascopy-node`, `luxon`, `trading-core-type` (github)
- **Visualization:** `vue`, `vuetify@4`, `@observablehq/plot`, `@mdi/font`

## Forbidden dependencies
- Higher-layer trading packages.

## Architecture
Monorepo: two packages (`core` + `visualizer`). Core implements fractal engine on candlestick data; visualizer builds web app (Vite + Vue3 + Vuetify4) and terminal demo. Vitest + Biome (lint/format).

## Distribution rule
- Core exports ESM (tsc → `dist/`), visualizer dual-mode (Node library + web build).
- Both use `workspace:*` protocol internally.
